RoomFinder is an Android application built using **Java** and **Android Studio**  
that helps users **find Rooms, PGs, Hostels, and Roommates** easily.

---

## Features
- Find Rooms, PGs, and Hostels
- Roommate Finder
- Location-based search (Google Maps API)
- Clean and simple UI
- Add/Save favorite listings

---

## Tech Stack
- **Language:** Java
- **IDE:** Android Studio
- **Database / Backend:** Firebase Realtime Database
- **Maps / Location:** Google Maps API
